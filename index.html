SymPy Combinatorics Tutorial

Introduction

Welcome to the SymPy Combinatorics tutorial! This guide will walk you through key concepts such as permutations and combinations using SymPy.

Permutations

Permutations involve rearranging elements in different ways. SymPy makes it easy to work with them.

Creating a Permutation

from sympy.combinatorics import Permutation

p = Permutation([2, 0, 1])
print("Permutation:", p)
print("Cycle Notation:", p.cyclic_form)

Output:

Permutation: Permutation([2, 0, 1])

Cycle Notation: [[0, 2, 1]]

Combinations

Combinations involve selecting elements without considering order.

Example

from sympy.functions.combinatorial.numbers import binomial

n, k = 5, 2
print("Combinations:", binomial(n, k))

Output:

Combinations: 10
